## 简介

** LIVE：一个基于WebRTC的可自建直播系统 **

<img src="https://i.loli.net/2021/11/24/r9GUSDqI1FVvLHo.png" alt="">

## 特色

- 基于WebRTC，毫秒级直播延迟；
- 可品牌化自营，最低成本建立属于自己的直播平台；
- 友好的界面（可定制UI）；
- 完全基于浏览器，即开即用；
- 可配置控制观众直播间进入权限；
- 多种推流方式，支持OBS，飞书等第三方推流工具。
